## Feature Details

### 1. XML Sitemap (`/sitemap.xml`)

**Location:** `frontend/src/app/sitemap.ts`

**Features:**
- Automatically generated by Next.js
- Includes all static pages with appropriate priorities:
  - Homepage: priority 1.0, daily updates
  - Product catalog: priority 0.9, daily updates
  - Gallery: priority 0.8, weekly updates
  - Contact: priority 0.7, monthly updates
- Dynamically fetches all products and includes product detail pages
- Sets `lastModified` based on product `updatedAt` timestamp
- Revalidates every hour (3600 seconds)

**Access:** `https://ebenor-creation.tn/sitemap.xml`

**Example Output:**
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://ebenor-creation.tn</loc>
    <lastmod>2024-01-15</lastmod>
    <changefreq>daily</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://ebenor-creation.tn/produits</loc>
    <lastmod>2024-01-15</lastmod>
    <changefreq>daily</changefreq>
    <priority>0.9</priority>
  </url>
  <url>
    <loc>https://ebenor-creation.tn/produits/cuisine-moderne-chene</loc>
    <lastmod>2024-01-10</lastmod>
    <changefreq>weekly</changefreq>
    <priority>0.8</priority>
  </url>
  <!-- More product URLs... -->
</urlset>
```

### 2. Robots.txt (`/robots.txt`)

**Location:** `frontend/src/app/robots.ts`

**Features:**
- Allows all public pages for all user agents
- Disallows admin pages (`/admin/`, `/admin/*`)
- Disallows API routes (`/api/`, `/api/*`)
- Disallows Next.js internal routes (`/_next/`)
- Includes sitemap URL
- Special rules for Googlebot and Bingbot

**Access:** `https://ebenor-creation.tn/robots.txt`

**Example Output:**
```
User-agent: *
Allow: /
Disallow: /admin/
Disallow: /admin/*
Disallow: /api/
Disallow: /api/*
Disallow: /_next/
Disallow: /private/

User-agent: Googlebot
Allow: /
Disallow: /admin/
Disallow: /admin/*
Disallow: /api/
Disallow: /api/*

User-agent: Bingbot
Allow: /
Disallow: /admin/
Disallow: /admin/*
Disallow: /api/
Disallow: /api/*

Sitemap: https://ebenor-creation.tn/sitemap.xml
```

### 6. Canonical URLs

**Requirements:** 23.5

All pages include canonical URL tags to prevent duplicate content issues:

```html
<link rel="canonical" href="https://ebenor-creation.tn/produits/cuisine-moderne-chene" />
```

Canonical URLs are:
- Absolute (include full domain)
- Consistent (always use the same URL format)
- Self-referential (each page points to itself)
- Clean (no query parameters unless necessary)

### 7. Semantic HTML Structure

**Requirements:** 23.9

All pages follow proper semantic HTML structure with correct heading hierarchy:

```html
<header>
  <nav>...</nav>
</header>

<main id="main-content">
  <article>
    <h1>Page Title</h1>
    <section>
      <h2>Section Title</h2>
      <h3>Subsection Title</h3>
    </section>
  </article>
</main>

<footer>...</footer>
```

**Heading Hierarchy Rules:**
- One `<h1>` per page (page title)
- Logical progression: h1 → h2 → h3 (no skipped levels)
- Descriptive headings that outline page structure
- Proper use of semantic elements: `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`

## Troubleshooting

### Sitemap Not Updating

**Problem:** Sitemap shows old products

**Solution:**
```bash
# Clear Next.js cache
rm -rf .next
npm run build
```

### Meta Tags Not Showing

**Problem:** Meta tags not visible in page source

**Solution:**
- Check if page is client-side rendered
- Use `useEffect` to update meta tags dynamically
- Verify `generateMetadata()` is called correctly

### Structured Data Errors

**Problem:** Google Rich Results Test shows errors

**Solution:**
1. Validate JSON-LD syntax
2. Ensure all required fields are present
3. Check data types match schema.org spec
4. Use Google's Structured Data Testing Tool

### Canonical URL Issues

**Problem:** Canonical URL points to wrong page

**Solution:**
- Check `NEXT_PUBLIC_SITE_URL` environment variable
- Verify `getCanonicalUrl()` function
- Ensure no trailing slashes
